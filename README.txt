
Run Snake Game on Eclipse: Click "Run" button on tab bar or Ctrl+ F11.

Snake Game's Rules:
   + User uses LEFT, RIGHT, UP, DOWN keys on the keyboard to move the snake to the LEFT, RIGHT, UP, DOWN respectively. 
   + Enemy has the random positions on the game board.
   + When snake hits enemy, snake's body increases by one dot.
   + When snake hits boundaries of the game board, Game is over.
